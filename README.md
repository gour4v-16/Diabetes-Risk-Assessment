# Diabetes Risk Assessment System

A machine learning based diabetes risk prediction platform.

## Frontend
- React
- Vite
- Tailwind CSS

## Backend
- FastAPI
- Python

## Machine Learning Models
- Random Forest
- XGBoost
- Artificial Neural Network (ANN)

## Dataset
- BRFSS 2023-2024
- 80,000 preprocessed records

## Deployment
- Frontend: Vercel
- Backend: Render
